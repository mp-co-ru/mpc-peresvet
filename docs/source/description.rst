Описание
--------
**МПК-Пересвет** - платформа для автоматизации технических объектов,
от агрегата до цеха. Также может быть использована для систем типа "умный дом".

Возможности: 

#. Создание моделей, включающих описание структуры объекта и его поведение.
#. Сбор данных, хранение, обработка, выполнение определённых действий 
   на базе происходящих событий. 
#. Автоматизация вычислений, бизнес-процессов, 
   происходящих в рамках моделируемого объекта.

Платформа может работать на большом спектре устройств: от одноплатных компьютеров
до кластеров серверов, поскольку может быть масштабирована.
