Установка

Опишем самый простой случай установки и запуска платформы.

Системные требования:

  1. Ubuntu 22.04

  2. Docker. Для Docker не забудьте выполнить настройку.

  3. Git.

Базовый способ развёртывания платформы - в виде docker-контейнеров.

  1. Клонируем проект платформы:

    $ git clone git@github.com:mp-co-ru/mpc-peresvet.git

  2. Заходим в каталог проекта:

    $ cd mpc-peresvet

  3. Запускаем платформу:

    $ ./run_one_app.sh

В результате будут запущены следующие компоненты платформы:

  1. Брокер сообщений RabbitMQ

  2. Служба каталогов OpenLDAP

  3. База данных PostgreSQL (данные будут храниться внутри контейнера)

  4. Ядро платформы

  5. Клиент для отображения данных Grafana

  6. Конфигуратор. Web-приложение для создания моделей.
