Компоненты

Компоненты платформы:

  1. Сервисы платформы.

    Исходный код платформы. Может быть запущен в виде одного приложения или как набор микросервисов.

  2. OpenLDAP.

    Иерархическая модель автоматизируемого объекта, модель информационной системы.

    Аутентификация и авторизация пользователей.

  3. RabbitMQ.

    Брокер сообщений. Микросервисы ядра общаются между собой с его помощью.

  4. Nginx.

    Прокси сервер, балансировщик нагрузки, TLS-прокси. Распределяет HTTP-запросы между микросервисами платформы, Grafan’ой и конфигуратором.

  5. Redis.

    Кэш-сервер.

  6. PostgreSQL.

    По умолчанию платформа содержит СУБД PostgreSQL в качестве исторической базы данных. В общем случае исторические данные могут храниться в нескольких базах, причём разных типов.

  7. Grafana.

    Инструмент для создания экранов мониторинга.

  8. Конфигуратор.

    Приложение для создания моделей объектов.
